# Data Analysis with Excel
Data Analysis with Microsoft Excel.

## Topics covered
1. Pivot Tables
2. Formulas
    * MAX and MIN
    * IF ans IFS
    * LEN
    * LEFT and RIGHT
    * TEXT
    * TRIM
    * CONCATENATE
    * SUBSTITUTE
    * SUM, SUMIF, and SUMIFS
    * COUNT, COUNTIF, and COUNTIFS

4. Charts
5. Data cleaning
